# Search Launch Checklist

## Google Search Console

1. Verify the preferred property for `https://www.designhavenbuild.com/`.
2. Submit `https://www.designhavenbuild.com/sitemap.xml`.
3. Inspect and request indexing for the priority URLs in `docs/indexing-priority.md`.
4. Confirm Google sees the canonical `www` URLs.
5. Review indexing, page experience, and query performance weekly.

## Bing Webmaster Tools

1. Verify `https://www.designhavenbuild.com/`.
2. Submit the XML sitemap.
3. Request indexing for the same priority URL list.

## Google Business Profile

1. Confirm the business name is exactly `Haven Design & Build LLC`.
2. Confirm phone, URL, service areas, hours, and address settings match what the owner wants public.
3. Add service links for kitchen remodeling, bathroom remodeling, basement finishing, decks/outdoor living, commercial renovation, Marietta, and Cobb County.
4. Upload genuine project photos only.
5. Request genuine client reviews after completed work.
6. Publish regular project/service updates using the templates in `/visibility/`.
7. Add Q&A answers for services, service area, decks, commercial work, payment schedule, and how to start.

## Measurement

1. Add GA4 measurement ID when supplied.
2. Connect GA4 to Search Console.
3. Track quote-form submissions, phone clicks, email clicks, service CTAs, and payment-tab clicks.
4. Preserve UTM parameters on campaign traffic where possible.

## Manual Citation / Backlink Work

1. Audit NAP consistency across business directories.
2. Claim or update relevant local profiles.
3. Use supplier/vendor/partner outreach templates in `/visibility/BACKLINK_OUTREACH_TEMPLATES.md`.
4. Track each submitted citation in `/visibility/CITATION_TRACKER.csv`.
