# Quick Start - Haven Design & Build

## What This Site Is

Static multi-section website with premium black + bronze theme for a luxury renovation business.

## Current Stack

- HTML: `index.html`
- CSS: `css/styles.css`
- JavaScript: `js/app.js`
- Config: `config.json`
- Assets: `images/`

## 10-Minute Setup

1. Open the site locally.
2. Replace Formspree form ID in `js/app.js`.
3. Submit a test inquiry.
4. Review Payments page behavior.
5. Deploy to static hosting.
