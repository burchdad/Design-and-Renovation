## 🌍 Deployment Options

The easiest paths to get your site live:

### Option 1: Netlify (EASIEST - Recommended)
- **Cost:** Free
- **Setup:** 2 minutes (drag & drop)
- **HTTPS:** Automatic
- **Domain:** Free subdomain, or add your own $10-15/year

### Option 2: Vercel
- **Cost:** Free
- **Setup:** 5 minutes (GitHub required)
- **HTTPS:** Automatic
- **Domain:** Free subdomain, or add your own

### Option 3: GoDaddy
- **Cost:** Domain $10-15/year + Hosting $4-10/month
- **Setup:** 20 minutes (upload via FTP)
- **HTTPS:** Included (AutoSSL)
- **Domain:** Can buy right there

**Recommendation for beginners:** Netlify - it's the fastest and easiest.

## 🆘 Troubleshooting Quick Links

| Issue | Solution |
|-------|----------|
| Form not submitting | Check Formspree Form ID in js/app.js |
| Images not loading | Verify image URLs are correct |
| Mobile menu not working | Clear cache, refresh browser |
| Payments not working | Stripe link must be valid URL |
| Calendly not showing | Check calendar is published in Calendly |
| Site loading slowly | Optimize image sizes |

See SETUP.md for detailed troubleshooting.
